# **TFS MultiplayerMod Server**
for client v1.1.0 (tfs v1.25.1)

(backup by kakaobob)
(author - vopwn55)


## Launch
Run:

```
python Server.py
```

Or use [Docker container](https://hub.docker.com/repository/docker/kakaobob/tfsmp-server).


## Config
Change `config.json`

## Plugins
You can enable ServersidePlugins by removing `!_` prefix in folder names
```
tfsmp-server\ServersidePlugins:
├ !_AdminDashboard
├ !_ExamplePlugin
├ !_Security Essentials 2024
└ !_ServerAnalytics
```
